# 3.2.4MultiValueMap&lt;K, V&gt;

```java

    void add(K var1, @Nullable V var2);

    void addAll(MultiValueMap<K, V> var1);

            this.add(key, value);
        }

    }

    void set(K var1, @Nullable V var2);

    void setAll(Map<K, V> var1);

    Map<K, V> toSingleValueMap();
```
